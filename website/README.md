# AI 应用工程师 — 24 周学习路线

## 使用方法

1. 双击 `index.html` 在浏览器中打开
2. 在"今日任务"页面设置开始日期（建议就设为今天）
3. 系统会自动计算每天该学什么，按天推进
4. 完成一项任务就点击打勾，进度自动保存到浏览器
5. 左侧导航切换：今日任务 / 完整课表 / 本周详情 / 学习资料库 / 实战项目 / 阶段自检 / 学习方法

## 网站功能

| 页面 | 说明 |
|------|------|
| 📊 今日任务 | 主页。显示今天该学什么，以及本周其他天数和任务 |
| 📚 完整课表 | 24 周全部课程，按月份分组，点击进入周详情 |
| 📅 本周详情 | 查看任意一周的每日任务，可前后翻周 |
| 🔗 学习资料库 | 精选的书籍/课程/工具/文档，按标签筛选 |
| 🛠️ 实战项目 | 三个旗舰项目的完整规格（要求/技术栈/交付物） |
| ✅ 阶段自检 | 6 个阶段的自检清单，确保每个阶段都达标 |
| 🧠 学习方法 | 6 位大神的方法论 + 10 条通用法则 + 每日行动清单 |

## 进度说明

- 进度数据保存在浏览器 localStorage 中
- 清除浏览器数据会丢失进度（建议不要清）
- 点击左下角"重置进度"可以重新开始

## 技术栈

纯静态 HTML/CSS/JS，无需安装任何东西，浏览器打开即用。
